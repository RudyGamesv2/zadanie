Poziom 3

CSS – kaskadowe arkusze stylów – język służący do programowania wyglądu stron opartych o język html.

Internal CSS

● Sposobem kodowania wyglądu strony jest zawarcie kodu css między tagami <style> i </style> Tagi te wstawiamy między tagami <head> i </head>. W kodzie tym odnosimy się do elementów zawartych w treści strony ● Do tagów html możemy odnosić się bezpośrednio np.h1 {color:red;} ● Lub poprzez klasy oraz identyfikatory (id). ● W tagach html wstawiamy atrybut class=”” lub id=”” ● Odwołanie się do klasy w css uzyskuje się za pomocą .nazwaklasy {kod css} ● Odwołanie się do identyfikatora id otrzymuje się za pomocą #nazwa_id {kod css} ● Różnica między klasami, a id jest taka, że klas używamy do elementów które wielokrotnie się powtarzają np. mamy wiele takich samych nagłówków, idy stosujemy, gdy chcemy zakodować wygląd pojedynczego elementu

Inline CSS

● Kod css możemy umieszczać w kilku miejscach. Pierwszy sposób to bezpośrednio w tagach html. ● Przykład: <p style="font-size: 200%;color: aquamarine">To jest powiększony i pokolorowany tekst</p> ● Jednak taki sposób kodowania wyglądu nie jest zalecany, ponieważ bardzo negatywnie wpływa na przejrzystość kodu, oraz musimy wiele razy wykonać te same czynności.

To jest powiększony i pokolorowany tekst

Oddzielne pliki z kodem CSS

● Ostatnim i najczęściej używanym sposobem dodawania wyglądu strony są oddzielne pliki, z rozszerzeniem .css ● Odnosimy się w nich do elementów strony podobnie jak z uzyciem tagu <style> ● Trzeba pamiętać o dodaniu odpowiedniego tagu dołączającego dany plik .css do strony html. Zawsze umieszczamy go w tagu <HEAD> <link rel="stylesheet" type="text/css" href="style.css">
